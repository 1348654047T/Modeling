Modeling
三维模型显示平台 目前支持以下功能：
1.模型 移动、旋转、缩放 
2.视图 移动、旋转、缩放 
3.支持常用三维模型的读入
4.允许一个三维模型里有很多的part
5.支持模型拾取，选中模型会显示包围盒
6.三种网格显示方式 实体显示、点显示、线显示 以及它们的组合显示

还需要做的： 目前这个三维模型管理平台，仅仅使用GLUT本身的功能，GLUT本省并不提供UI操作，因此交互不太好。UI方面考虑使用MFC或者QT，MFC 较熟悉，但还是感觉很难用，可能考虑使用QT进行做UI整合进来。

使用须知： 这个工程是在windows平台VS2010下进行编译的，使用了opengl 的GLUT库，使用的时候下载这个工程到本地，用VS2010或更高版本打开.vcxproj 工程文件，并配置好GLUT库文件跟头文件即可正常使用
